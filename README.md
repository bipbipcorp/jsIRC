ChatJS
======

IRC 2.0 (Json based Web IRC) implementation, using Node.js, Socket.io, Extjs, Phantomjs and jQuery.

Based on RFC2812: http://tools.ietf.org/html/rfc2812.

Supported Client to Server commands:

- NICK
- USER
- WHOIS
- JOIN
- PART
- PRIVMSG
- MOTD
- LUSERS
- PONG
- TOPIC
- LIST
- OPER
- NAMES
- QUIT
- AWAY
- MODE
- KILL
- INFO
- ADMIN
- TIME
- VERSION
